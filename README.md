# health-care-ai
# 🩺 Smart Healthcare Monitoring & AI Consultation Platform

A modern healthcare web application that combines health monitoring, doctor appointment booking, medicine ordering, AI-based health suggestions, and emergency healthcare support into a single platform.

---

# Features

✅ Health Monitoring Dashboard
✅ ECG-Style Animated UI
✅ Doctor Appointment Booking
✅ Online & Offline Consultation
✅ Medicine Ordering System
✅ AI-Based Health Suggestions
✅ Emergency Healthcare Support
✅ Health Records Management
✅ Medical Timeline Tracking
✅ Responsive Modern UI

---

**# Installation & Setup**

## Clone Repository

```bash
git clone https://github.com/manaswini-2403/health-care-ai.git
```

## Install Dependencies

```bash
npm install
```

## Run Project

```bash
npm start
```
